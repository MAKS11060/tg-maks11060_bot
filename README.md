| ------- | -------------------- |

### Inline commands

#### [Danbooru](https://danbooru.donmai.us)

```ts
// daily top
@maks11060_bot hot

// user favorites (me)
@maks11060_bot fav

// saved searches
@maks11060_bot saved

// get pic by id
@maks11060_bot id 8660130 8442841 7058195 1
```
